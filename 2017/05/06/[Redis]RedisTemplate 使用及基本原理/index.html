  • 连接池配置对象
  • Jedis连接工厂，提供 getConnection 方法从连接池中获取连接

思路：1、首先必须有 Redis 操作类 2、然后为了提高性能，加入 Redsi 连接池，调用连接池的 getConnection 方法获得连接
